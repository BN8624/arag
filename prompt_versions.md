# Prompt Versions — 프롬프트 실험 장부

`prompts.PROMPT_VERSION`의 변경 이력과 측정 결과. **프롬프트(prompts.py 또는
reviewer.py)를 바꾸면 반드시: ① PROMPT_VERSION 갱신 ② 여기 항목 추가.**
런마다 이 버전이 `runs/index.json`에 기록되므로, 전후 비교의 분모는 이 장부가 가른다.

운영 규칙:
- 배치당 프롬프트 변경 1개 (변수 격리)
- 판정은 느낌이 아니라 지표로: OK율 / 평균 점수 / NOCHANGE율 / perfect-but-gap /
  improve 판정 분포 / 재발률 / 비용. 단 **level(출제 난이도)과 인프라 장애(ERROR)를
  분리하고 볼 것** — 성공률 하락이 프롬프트 탓이 아닐 수 있다
- 측정 도구: `python analyze_batch.py` (콜 0) + 대시보드 지표 탭
